# CodSoft-DS-Internship---Titanic-Survival-Prediction
# 🚢 Titanic Survival Prediction using Machine Learning

## CodSoft Internship – Task 1

This project is developed as Task 1 of the Data Science Internship offered by [CodSoft](https://www.codsoft.in?utm_source=chatgpt.com). The objective of this task is to build a Machine Learning model that predicts whether a passenger survived the Titanic disaster based on different passenger attributes.

---

# 📌 Project Overview

The Titanic Survival Prediction project is based on the famous Kaggle Titanic dataset. The goal is to predict passenger survival using Machine Learning classification algorithms.

This project demonstrates the complete Machine Learning workflow including:

* Data Collection
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training
* Model Evaluation

The project is implemented using Python in Google Colab.

---

# 📂 Dataset Information

The dataset contains passenger information such as:

* Passenger Class (`Pclass`)
* Name
* Gender (`Sex`)
* Age
* Fare
* Number of Siblings/Spouses (`SibSp`)
* Number of Parents/Children (`Parch`)
* Embarked Port
* Survival Status (`Survived`)

Dataset Source:
[Kaggle Titanic Dataset](https://www.kaggle.com/competitions/titanic/data?utm_source=chatgpt.com)

# 📊 Project Workflow

## 1️⃣ Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature selection
* Data normalization/scaling

## 2️⃣ Exploratory Data Analysis (EDA)

* Survival count analysis
* Gender-wise survival comparison
* Passenger class analysis
* Correlation heatmaps
* Data visualization using charts and graphs

## 3️⃣ Model Building

Machine Learning algorithms used:

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

## 4️⃣ Model Evaluation

The model performance is evaluated using:

* Accuracy Score
* Precision
* Recall
* F1-Score
* Confusion Matrix

---

# 📈 Results

The trained Machine Learning model successfully predicts passenger survival with good accuracy after preprocessing and feature engineering. Among different algorithms, Random Forest and Logistic Regression provide effective results for this dataset.
